HELLO PAINT - TEMPLATES & TOOLKIT
=================================

The working folder. Everything here is a starting point you edit, not a
finished asset. Open any file, swap the [bracket] placeholders for real
words or photos, and you have a kit insert, a label, a post, or a signature
that already matches the brand.

Static, finished versions of these pieces live in the asset library
(05-asset-library). The files in here are the editable cousins: they keep the
fill-in fields and the guide notes so you never start from a blank page.

HOW TO EDIT THE SVG TEMPLATES
-----------------------------
Every .svg here is plain text and fully editable. You have three easy paths:

  Canva     Upload the SVG (Create a design > Upload). Click any text to retype.
            Click the dashed photo frames and drop in an image.
  Figma /   File > Place image, or just open the SVG. All text stays as live
  Illustrator  text you can click and edit. Colors are named hex values.
  Quick peek   Double-click the SVG to open it in a browser and see it. To
            edit, use one of the tools above (a browser only views it).

The fill-in spots are written as [first name], [your note here], no. 0042,
SHIP TO, and the dashed photo frames. Replace those and leave the rest alone.

Fonts: the templates call for Fraunces (headings) and Nunito Sans (body and the
wordmark). Both are in 04-typography. If a tool does not have them, install the
font files first, or it will fall back to Georgia and Trebuchet MS, which are
the built-in stand-ins already named in each file.


EMAIL SIGNATURE - the short version
-----------------------------------
  1. Double-click email-signature.html. It opens in your browser.
  2. Click any highlighted field (name, role, email, links) and type over it.
  3. Select the whole card inside the dashed box and copy it.
  4. Paste into Gmail (Settings > See all settings > Signature) or Apple Mail
     (Settings > Signatures). The colors, the wave, and the dots come with it.
  5. One wave only, no extra emoji. That is the rule.


BRAND QUICK RULES (so edits stay on brand)
------------------------------------------
  Colors    Raspberry #E64C81   Teal #15A39A   Sunshine #F6C744
            Leaf #5BA86B        Plum ink #312B3D (the dark; use instead of black)
            Cream #FBF4E8 (the background)
  Type      Fraunces for headings, Nunito Sans for body and the wordmark.
  Voice     Cozy and plain. Invite, do not instruct. "your photo, painted by you"
  Avoid     Orange, Title Case or all-caps wordmark, gradients, shadows, and
            body text in bright raspberry or teal.


KEY (how this note is written)
------------------------------
  UPPERCASE HEADINGS   Section breaks.
  [bracket text]       A blank to fill in inside a template.
  Indented mono rows   Exact values to copy: file names, hex codes, sizes.
  Hyphen "-"           Used in place of a dash on purpose. No em dashes, because
                       they read as minus signs in STEM contexts.


GLOSSARY
--------
  SVG          A graphic that stays sharp at any size (vector). Plain text
               inside, so the words and colors are editable. Opens in Canva,
               Figma, Illustrator, or any browser.
  Template     A starting file you edit, not a finished asset.
  Placeholder  The [bracket] text or dashed frame that marks where to add yours.
  Wordmark     The "hello paint" logo made from styled text.
  Mark         A small symbol logo (here: the smile-and-sparkle, the snapshot).
  CTA          Call to action. The button-style prompt, like "shop now".
  Hex          A color code like #E64C81. The exact brand colors.
