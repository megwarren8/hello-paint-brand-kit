# Hello Paint: brand system

The brand system I designed and built for Hello Paint, my own custom
paint-by-number shop. Live at
**[hellopaint.megan-warren.com](https://hellopaint.megan-warren.com)**.

Designed and built by [Megan Warren](https://megan-warren.com).

## What's here

- **The brand book** (`01-brand-book.html`, also the site's landing page):
  the full identity in narrative form, voice, color, type, and marks, end to
  end, offline-capable.
- **A numbered asset library**: logos and marks, color specs, embedded
  Fraunces and Nunito Sans fonts, social templates, stationery, icons,
  stickers, and motion references, plus an editable templates and toolkit
  folder.
- **00-START-HERE** as the quick-reference dashboard for the whole kit.

## Reading it locally

Open **[index.html](index.html)** in any browser; the kit works from a plain
folder with no internet, no build step, and no dependencies.

## How it deploys

Every push to `main` republishes the site to Cloudflare Pages via
`.github/workflows/deploy.yml`.

## About

Not a stock brand template. A one-off system, made for one shop. All rights
reserved.
